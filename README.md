# Tailwind JIT Starter React

Feel free to use the sample code to setup tailwind with jit for your app.

## Setup

1. Clone repo

```bash
git clone https://github.com/evan70/react-tailwind-jit-just-in-time-starter.git
```

2. Install Packages

```bash
npm i
```

3. Watch Css

```bash
npm run watch:css
```

4. Start Server

```bash
npm start
```
